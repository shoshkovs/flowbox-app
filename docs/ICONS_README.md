# Иконки для навигации

## Куда поместить файлы

Поместите ваши PNG иконки в эту папку с следующими именами:

- `menu.png` - иконка для вкладки "Меню"
- `cart.png` - иконка для вкладки "Корзина"
- `profile.png` - иконка для вкладки "Профиль"
- `about.png` - иконка для вкладки "О проекте"

## Рекомендуемые размеры

- **Размер:** 24x24px или 48x48px (для Retina дисплеев)
- **Формат:** PNG с прозрачным фоном
- **Стиль:** 
  - Обычное состояние: серый цвет (#999)
  - Активное состояние: розовый цвет (#ff6b9d)

## Варианты

### Вариант 1: Одна иконка на вкладку
Просто поместите файлы с указанными именами.

### Вариант 2: Две иконки (обычная и активная)
Если хотите разные иконки для активного состояния, используйте:
- `menu.png` и `menu-active.png`
- `cart.png` и `cart-active.png`
- и т.д.

В этом случае нужно будет обновить JavaScript для переключения иконок.

## Пример структуры

```
public/
  └── icons/
      ├── menu.png
      ├── cart.png
      ├── profile.png
      └── about.png
```

## Fallback

Если PNG файлы не найдены, автоматически будут использованы SVG иконки как резервный вариант.
