# Настройка деплоя на Render.com

## Build команда

На Render.com в настройках сервиса установи:

**Build Command:**
```bash
npm run build
```

⚠️ **ВАЖНО:** Не используйте `npm install` - это только установит зависимости, но не соберет админку!

Текущая команда `npm run build` делает:
1. Переходит в папку `admin`
2. Устанавливает зависимости админки (`npm install`)
3. Собирает админ-панель (`npm run build`)

## Start команда

**Start Command:**
```bash
npm start
```

Эта команда просто запускает `node bot.js` без сборки.

## Как изменить в Render.com

1. Откройте настройки сервиса (Settings)
2. Найдите секцию **"Build Command"**
3. Нажмите кнопку **"Edit"** (иконка карандаша)
4. Измените команду с `npm install` на `npm run build`
5. Сохраните изменения

## Важно

- Сборка админки должна происходить **только** во время build фазы
- Если `admin-build` не существует при запуске, сервер будет использовать fallback (исходники)
- Убедитесь, что build команда выполняется **до** start команды

## Проверка

После деплоя проверьте логи:
- Должно быть: `✅ Админ-панель: React версия из admin-build/`
- Не должно быть: `⚠️  admin-build не найден, выполняем сборку...`
