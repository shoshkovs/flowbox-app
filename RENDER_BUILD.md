# Настройка деплоя на Render.com

## Build команда

На Render.com в настройках сервиса установи:

**Build Command:**
```bash
npm run build
```

Эта команда:
1. Устанавливает зависимости админки (`cd admin && npm install`)
2. Собирает админ-панель (`npm run build` в папке admin)

## Start команда

**Start Command:**
```bash
npm start
```

Эта команда просто запускает `node bot.js` без сборки.

## Важно

- Сборка админки должна происходить **только** во время build фазы
- Если `admin-build` не существует при запуске, сервер будет использовать fallback (исходники)
- Убедитесь, что build команда выполняется **до** start команды

## Проверка

После деплоя проверьте логи:
- Должно быть: `✅ Админ-панель: React версия из admin-build/`
- Не должно быть: `⚠️  admin-build не найден, выполняем сборку...`
