# Настройка React админ-панели

## Быстрый старт

1. **Установите зависимости админ-панели:**
```bash
cd admin
npm install
```

2. **Соберите админ-панель для production:**
```bash
npm run build
```

3. **Запустите сервер:**
```bash
cd ..
npm start
```

Админ-панель будет доступна по адресу: `http://localhost:3000/admin`

## Разработка

Для разработки с hot-reload:

```bash
cd admin
npm run dev
```

Админ-панель будет доступна по адресу: `http://localhost:5173`

## Структура

- `admin/src/` - исходный код React приложения
- `admin/src/components/` - компоненты страниц
- `admin-build/` - собранная версия (создается после `npm run build`)

## Авторизация

По умолчанию пароль: `admin123`

## Примечания

- После изменений в админ-панели необходимо пересобрать проект (`npm run build`)
- Express сервер автоматически использует собранную версию из `admin-build/`
- Если сборка отсутствует, используется старая HTML версия из `admin/`
